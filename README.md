# Smart Cable Inspection System – ElSewedy Case Study

## 📌 Overview

The **Smart Cable Inspection System** is an intelligent solution developed as part of a case study for Elsewedy Electric.
The project aims to automate the inspection process of industrial cables using **Computer Vision**, reducing reliance on manual inspection and improving defect detection accuracy.

The system integrates image processing, machine learning, and a user-friendly GUI to provide a complete end-to-end inspection pipeline.

---

## 🚀 Features

* 🔍 **Automated Defect Detection** using Computer Vision
* 🧠 **Image Processing & Classification Models** for cable surface analysis
* 💻 **Interactive GUI** for real-time monitoring and control
* ⚡ **Fast & Efficient Inspection Pipeline**
* 📊 **Visual Output of Results** for better decision-making

---

## 🧠 Computer Vision Module

The core of the system is built on **Computer Vision techniques**, which:

* Capture and process cable images
* Detect surface defects (e.g., cuts, irregularities, damages)
* Extract important visual features
* Classify cable conditions with high accuracy

This significantly enhances quality control by minimizing human error and increasing consistency.

---

## 🏗️ System Architecture

The system is composed of three main components:

1. **Input Layer**

   * Image acquisition from real production line / live demo

2. **Processing Layer**

   * Image preprocessing (filtering, resizing, enhancement)
   * Feature extraction
   * Defect detection & classification

3. **Output Layer**

   * Visualization through GUI
   * Display of detected defects and status

---

## 🖥️ GUI Application

A custom-built GUI was developed to:

* Display real-time inspection results
* Visualize detected defects
* Provide an easy-to-use interface for operators
* Improve usability in industrial environments

---

## 📊 Results

* Achieved high defect detection accuracy (**~90–95%**)
* Reduced manual inspection effort (**~60%**)
* Improved monitoring and response time (**~40% faster**)
